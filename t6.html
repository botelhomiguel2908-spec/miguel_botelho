Introdução ao CSS

CSS (Cascading Style Sheets) é uma linguagem utilizada para definir o estilo de documentos HTML. Ele permite controlar cores, fontes, espaçamentos, posicionamentos e diversos aspectos visuais de uma página web.

Com CSS, é possível separar o conteúdo da página (HTML) da sua apresentação visual, tornando o código mais organizado e facilitando a manutenção.

Existem três formas principais de aplicar CSS a uma página HTML:

  • Inline: diretamente no elemento HTML com o atributo style.
  • Interno: dentro de uma tag <style> no cabeçalho do HTML.
  • Externo: em um ficheiro .css separado, ligado ao HTML com <link>.

Exemplo de CSS interno para mudar a cor do texto:

p {
  color: blue;
  font-size: 16px;
}

CSS é essencial para tornar páginas web visualmente atraentes e melhorar a experiência do usuário, permitindo designs responsivos e modernos.
