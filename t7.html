Seletores CSS

Os seletores CSS são utilizados para identificar quais elementos HTML irão receber determinado estilo. Eles permitem aplicar regras de formatação de forma precisa e organizada.

Existem vários tipos de seletores em CSS, sendo os mais comuns os seletores de elemento, de classe e de ID.

Seletor de Elemento: seleciona todos os elementos HTML de um mesmo tipo.

p {
  color: green;
}

O código acima aplica a cor verde a todos os parágrafos da página.

Seletor de Classe: utiliza o símbolo . e pode ser reutilizado em vários elementos.

.destaque {
  background-color: yellow;
  padding: 10px;
}

Classes são ideais quando se deseja aplicar o mesmo estilo a vários elementos diferentes.

Seletor de ID: utiliza o símbolo # e deve ser único na página.

#titulo {
  color: blue;
  text-align: center;
}

O ID é usado quando se pretende estilizar um elemento específico.

Além destes, existem outros seletores importantes:

  • Seletor Universal (*): seleciona todos os elementos da página.
  • Seletor Descendente: seleciona elementos dentro de outros.
  • Seletor de Atributo: seleciona elementos com atributos específicos.
  • Pseudo-classes: definem estilos para estados especiais, como :hover.

O uso correto dos seletores CSS torna o código mais limpo, organizado e facilita a manutenção e evolução do site.
